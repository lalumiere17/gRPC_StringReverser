gRPC string reverser service (Node.js)
===========================

`node`: This requires Node 0.12.x or greater.

Install
-------

   ```sh
   $ # Run npm install to install node modules from package.json
   $ npm install
   ```

Start
-------

 - Run the server

   ```sh
   $ # from this directory
   $ node ./server.js
   ```

 - Run the client

   ```sh
   $ # from this directory
   $ node ./client.js
   
   OR running with params
   
   $ node ./client.js stringToReverse
   ```
